How to Analyze Performance

$ cd someDirectory/
$ git clone https://github.com/g1257/PsimagLite.git
$ git clone https://github.com/g1257/dmrgpp.git
$ cd dmrgpp/src
$ perl configure.pl < ../TestSuite/inputs/callgrind.spec
$ make
$ valgrind --tool=callgrind ./dmrg -f ../TestSuite/inputs/input3.inp

Discovering Memory Issues

$ perl configure.pl < ../TestSuite/inputs/debug.spec
$ make
$ valgrind --tool=memcheck ./dmrg -f ../TestSuite/inputs/input3.inp

Discovering Race Conditions

I'm working on building a drd.spec file

$ perl configure.pl < ../TestSuite/inputs/drd.spec
$ make
$ valgrind --tool=drd ./dmrg -f ../TestSuite/inputs/input3.inp
